# 📰 Fake News Detection System

A Machine Learning-based web application that detects whether a news article is **Fake** or **Real** using Natural Language Processing (NLP) and a Passive Aggressive Classifier.

## 🚀 Features

- 🔍 Detect Fake or Real News
- 📊 Confidence Score
- 📝 Prediction History
- 👤 User Authentication
- 📱 Responsive UI
- 🌙 Dark Mode
- 📈 Admin Dashboard
- 🔄 Model Retraining
- 🌐 REST API
- 🗄 SQLite Database
